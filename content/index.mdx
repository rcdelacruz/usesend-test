# useSend Testing Suite

A comprehensive testing suite for your self-hosted useSend instance. Verify your configuration, test email delivery, and ensure everything works perfectly.

## Get started in 30 seconds

### 1. Install Dependencies

```bash
npm install
```

### 2. Configure Environment

Copy the example environment file and add your credentials:

```bash
cp .env.example .env
# Edit .env with your useSend API key and settings
```

### 3. Run Tests

```bash
npm test
```

That's it! Continue with [Getting Started →](/getting-started)

---

## What this suite does for you

**Test your configuration**: Automatically validates all environment variables and ensures your useSend instance is properly set up.

**Verify email delivery**: Send test emails via REST API and SMTP to confirm your email service is working correctly.

**Debug issues**: Get detailed error messages and troubleshooting guidance when something goes wrong.

**Monitor AWS SES**: Understand sandbox limitations and get step-by-step instructions for production access.
